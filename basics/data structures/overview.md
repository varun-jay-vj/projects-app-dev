## Program

Input --> || Program || --> Output

Client (User) || <<--Response--- --Request--->> || Server (Program)

Programs take input (device, files, database) and compute an output based on a set of instructions specified. 
Python is a high level programming language (foundations are built using C, which is a mid-level programming language).

Program can contain math operations, conditional executions or repetitions.

**Programming = Debugging**
Error Types
- Syntax errors: doesn't follow the structure & tokens defined in the formal language of Python
- Runtime errors: exceptions that are encountered only while the interpreter runs the program
- Semantic errors: errors that don't stop program execution but might not generate the output you desire as a developer.
